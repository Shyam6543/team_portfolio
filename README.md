# GANTS — Team Portfolio

This is the static portfolio site for Team GANTS. It's a simple HTML/CSS/JS site ready to be hosted as a static site (GitHub Pages, Netlify, Vercel, etc.).

Quick local preview:

```bash
npm install -g http-server   # optional
npm start
# or
npx http-server -c-1 . -p 8080
```

Deploy to GitHub Pages:

1. Push this repository to GitHub.
2. The included GitHub Actions workflow will publish the site automatically from the default branch to GitHub Pages.
